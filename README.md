# Code Aggregator

Приложение для сбора текстовых файлов из проектов в единый документ. Удобно для быстрого создания контекста кода для LLM, резервного копирования логики или поиска по всему проекту.

## Что делает

- **Сканирует** указанные папки и файлы, фильтрует по расширениям
- **Собирает** текстовое содержимое в один файл формата: `имя_файла:"содержимое"`
- **Автообновляет** результат при изменении исходных файлов
- **Бекапит** результат автоматически и вручную
- **Работает** на Windows и macOS

## Стек

- Python 3.11
- CustomTkinter (тёмный GUI)
- Watchdog (файловый мониторинг)
- Pillow (обработка иконок)
- PyInstaller (сборка .exe)

## Установка

```bash
# Клонирование
git clone https://github.com/ingaar02/code-aggregator.git
cd code-aggregator

# Windows
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

# macOS
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
python src/main.py
```

## Сборка в .exe (Windows)

```bash
pyinstaller --onefile --windowed --name="CodeAggregator" src/main.py
```

## Функционал

| Фича | Описание |
|------|----------|
| Менеджер проектов | Несколько проектов с разными настройками |
| Автоопределение стека | Определяет React, Python, Kotlin и др. по файлам |
| Фильтр расширений | Чипсы с добавлением через Enter |
| Файловый мониторинг | Автопересборка при изменениях |
| Автобекапы | Каждые 5 мин при изменениях, ротация 10 шт. |
| Ручные бекапы | Сохраняются навсегда |
| Кроссплатформа | Windows + macOS |

## Структура

```
code-aggregator/
├── src/
│   ├── core/          # Логика: агрегация, бекапы, проекты
│   ├── gui/           # Интерфейс: редактор, сайдбар, компоненты
│   └── main.py        # Точка входа
├── config/            # Настройки и проекты (не в Git)
├── output/            # Результаты и бекапы (не в Git)
└── requirements.txt
```
