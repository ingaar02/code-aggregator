# Code Aggregator

Приложение для сбора текстовых файлов из проектов в единый документ. Удобно для быстрого создания контекста кода для LLM, резервного копирования логики или поиска по всему проекту.

## Что делает

- **Сканирует** указанные папки и файлы, фильтрует по расширениям и исключает бинарные файлы
- **Собирает** текстовое содержимое в один файл формата: `имя_файла:"содержимое"` (с экранированием кавычек, слэшей и переносов строк)
- **Автообновляет** результат при изменении исходных файлов (сравнение по MD5-хешу)
- **Бекапит** результат автоматически (каждые 5 мин при изменениях, ротация 10 шт.) и вручную (сохраняются навсегда)
- **Управляет проектами** — несколько проектов с независимыми настройками, иконками, алиасами
- **Интегрируется с Git** — Pull и Push (с настраиваемым сообщением коммита) из первого источника проекта
- **Определяет стек** технологий по содержимому папки (React, Vue, Python, Kotlin, Go, Rust и др.)
- **Работает** на Windows и macOS (кроссплатформенное открытие файлов/папок)

## Стек

- Python 3.11+
- CustomTkinter (тёмный GUI, масштабирование 1.25x)
- Pillow (обработка иконок: кадрирование в квадрат, ресайз до 512×512)
- PyInstaller (сборка .exe)
- Стандартная библиотека: `threading`, `subprocess`, `hashlib`, `pathlib`, `dataclasses`

## Установка

```bash
# Клонирование
git clone https://github.com/ingaar02/code-aggregator.git
cd code-aggregator

# Windows
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

# macOS
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
