# Changelog

## [1.0.0] - 2026-06-18

### Добавлено
- Менеджер проектов с автоопределением стека технологий
- Фильтрация файлов по расширениям (интерактивные чипсы)
- Автоматическая пересборка при изменении исходных файлов
- Автоматические бекапы (ротация 10 шт., интервал 5 мин)
- Ручные бекапы с пометкой
- Git Pull/Push из интерфейса
- Обработка иконок проектов (кадрирование, сжатие до 512×512)
- Тёмная тема интерфейса (CustomTkinter)
- Кроссплатформенная поддержка: Windows (.exe установщик), macOS (.dmg), Linux (.AppImage)
- Тултипы на всех элементах интерфейса
- Прогресс-бар для длительных операций
- Копирование содержимого бекапов в буфер обмена
- Открытие папок и файлов в системном проводнике

### Технические детали
- Python 3.11 + CustomTkinter 5.2.2
- PyInstaller 6.0.0 для сборки исполняемых файлов
- Inno Setup — профессиональный установщик Windows
- DMG образы для macOS
- AppImage для Linux (без установки в систему)
